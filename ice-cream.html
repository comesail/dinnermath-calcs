Quick Notes

  • Typical scoop: ~½ cup (~4 fl oz). Pint ≈ 4 scoops • Quart ≈ 8 • Half-gallon ≈ 16 • Gallon ≈ 32.
  • Baseline: ~1.3 scoops per adult-equivalent for a standard “single.” Use Double for hearty servings.
  • Sundae bars: Extras disappear quickly—consider “Some leftovers.”
  • Variety: Split tubs across flavors; a small buffer helps keep every flavor available.

FAQs

How many scoops per person?
Plan about 1.3 scoops per adult-equivalent for a standard single; select Double (2.0) for generous servings.
How many pints/quarts/gallons do I need?
We convert total scoops to containers using typical yields: Pint ≈ 4, Quart ≈ 8, Half-gallon ≈ 16, Gallon ≈ 32 scoops.
What about many flavors or toppings?
Add a small buffer (Leftovers: Some) so each flavor lasts through the event and sundae bars don’t run short.
