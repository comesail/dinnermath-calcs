Mini FAQ

How many meatballs per person?
As a main, plan about 5–7 medium meatballs per adult (3–4 for teens, 2–3 for kids). As an appetizer, about 2–3 per adult. This calculator adjusts for appetite, event length, and menu variety.
What size does this assume?
Default is ~1.5 oz cooked per meatball. Change the size if yours are smaller or larger; pounds will update automatically.
Beef, pork, turkey—does it change the math?
Flavor blend doesn’t change counts. Only size, appetite, and event context affect totals.
Do the results include sauce?
Weights are for meatballs only. Add sauce separately if you’re budgeting by weight.
